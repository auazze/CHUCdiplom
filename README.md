## Раздел 1: Установка окружения

Всё ПО бесплатное. Устанавливать в порядке списка.

### 1. Visual Studio 2022 Community
**Скачать:** https://aka.ms/vs/17/release/vs_community.exe

При установке выбрать **одну** рабочую нагрузку:
> ✅ Разработка классических приложений .NET

Всё остальное — не трогать. Размер установки ~6 ГБ.

---

### 2. SQL Server 2022 Developer
**Скачать:** https://go.microsoft.com/fwlink/p/?linkid=2215158

При установке выбрать тип: **Basic**. Нажал — готово.

После установки запомнить строку подключения:
```
Server=localhost;Database=master;Trusted_Connection=True;
```

---

### 3. SQL Server Management Studio (SSMS)
**Скачать:** https://aka.ms/ssmsfullsetup (~500 МБ, полный установщик)

Установка: Далее → Далее → Установить. Никаких настроек.

**Если при подключении ошибка про сертификат:**
В окне Connect to Server → Options >> → Connection Security → поставить галочку **Trust server certificate**

---

### 4. draw.io Desktop
**Скачать:** https://github.com/jgraph/drawio-desktop/releases

Скачать последний `.exe` файл. Используется для построения ER-диаграмм (Модуль 1).
Работает полностью оффлайн — важно для экзамена.
